


###########################################Description des scripts pour les enjeux #############################################



Tous les fichiers.py de ce dossier présentent la même structure. Je la décris une fois dans le fichier "antisémitisme". 


J'ai utilisé les librairies Pandas, et Numpy du langage Python (dernière version). L'écriture du programme a été réalisé sur un notebook jupyter, 
émulé par Anaconda Navigator, et dans l'environnement conda.
Pour l'utiliser, installer Python sur l'ordinateur, l'environnement conda puis l'application Anaconda Navigator.Choisir "Jupyter Notebook". 

Pour plus d'informations, voir:

https://www.python.org/
